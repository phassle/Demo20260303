Agentic Development Workshop

      
This project is a teaching codebase for the Agentic Development Workshop (March 3-4, 2026). Developers learn to work with AI coding agents — primarily Claude Code — to identify bugs, improve code quality, and follow Rails best practices.

      
The Rails API contains 5 intentional issues that serve as exercises. Participants use AI agents to diagnose and fix each issue, following a structured workflow:

      

        

          
1

          
Plan

          
Understand the issue and design a fix

        

        

          
2

          
Build

          
Implement the solution with TDD

        

        

          
3

          
Simplify

          
Refactor for clarity and convention

        

        

          
4

          
Verify

          
Run tests and review edge cases
